# Магазин одежды: MATE

Сайт магазина одежды MATE. Можно просматривать коллекции, закидывать интересующие позиции в корзину, просматривать текущие покупки. Также на сайте присутствует возможность создания личного кабинета где можно ознакомиться с завершенными и текущими покупками, их датой и местом проведения.

Github Pages: https://katokinawa.github.io/MATE-Clothing-Store/
